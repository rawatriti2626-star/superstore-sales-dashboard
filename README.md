# superstore-sales-dashboard
## Project Overview

Developed an interactive Power BI dashboard to analyze retail sales performance, product trends, and regional insights. The dashboard helps visualize key business metrics and supports data-driven decision-making.

## Tools & Skills Used

* Power BI
* Data Visualization
* Data Cleaning
* Dashboard Design
* Basic DAX
* SQL Concepts

## Key Insights

* West region led total sales
* Technology was the top-performing category
* Sales increased steadily after 2015
* Year-end months showed stronger sales trends
